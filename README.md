此服务器框架使用C语言Libevent库，运行平台为Linux，使用GCC编译器

本项目有自动化编译脚本makefile
在控制台中执行make来编译，在obj文件夹中生成.o中间文件，在根目录生成server.out可运行文件
执行make clean来清理.o中间文件

可在根目录配置文件server.conf中修改服务器资源目录，如下
root_dir=/home/lelqwq/server_resources/
如果指定的目录无效，则会生成新的目录
